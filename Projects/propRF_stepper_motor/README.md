# Proprietary RF Echo RX/TX for Stepper Motor Control

---

Description
-----------
These two folders are companion projects for the CC2340R5:

* rfEchoRx_LP_EM_CC2340R5_freertos_ticlang: Default rfEchoRx project with minor modifications for rotate commands to the stepper motor on the separate device
* rfEchoTx_stepper_motor: Stepper motor design with both local or radio control
